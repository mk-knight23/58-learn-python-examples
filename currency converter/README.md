# Currency Calculator

A dynamic currency converter that fetches real-time exchange rates to provide accurate conversion calculations.

## Features

- Real-time exchange rate fetching
- User-friendly GUI with PyQt5
- Support for multiple currencies
- Automatic rate updates

## Tech Stack

- **Python 3.8+**
- **GUI Framework**: PyQt5
- **Web Scraping**: BeautifulSoup4, Requests
- **Exchange Rates**: Free API integration

## Installation

1. Clone this repository:
   ```bash
   cd currency\ converter
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

   Or individually:
   ```bash
   pip install bs4 requests pyqt5
   ```

## Usage

1. Run the application:
   ```bash
   python currency_converter.py
   ```

2. Enter the amount to convert
3. Select source and target currencies
4. View the converted amount

## Requirements

- Python 3.8 or higher
- Internet connection for fetching exchange rates
